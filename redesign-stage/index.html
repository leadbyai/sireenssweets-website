Reference set

  1. Milk BarBold utility nav, promo strip, conversion-forward product rows, high-contrast CTAs.
  2. Levain BakeryCream + cobalt identity, oversized product photography, playful badges, very clear “shop” path.
  3. Magnolia BakeryOccasion/category ordering, nostalgic serif moments, strong local pickup/delivery routing. Caveat: modal/cookie clutter can obscure content.
  4. Dominique AnselChef-led editorial credibility, full-bleed hero, visual navigation tiles for major journeys.
  5. Porto's BakeryDecision-first CTA panel: pickup, bake at home, catering; strong location/menu utility.
  6. SusieCakesReference could not be visually evaluated because Cloudflare bot verification blocked headless capture; included for category relevance only.
  7. Flour Bakery + CafeWarm story/owner personality, strong order button, human local voice. Caveat: popup/cookie layers reduced first impression.
  8. Tartine BakeryMinimal artisan/editorial posture; headless capture showed partial loading, so use as craft/minimal reference not layout source.
  9. Supermoon BakehouseHigh-fashion bakery identity: dramatic type, black/pink contrast, huge wordmark, memorable art direction.
  10. Baked by MelissaBright seasonal hero, horizontal category/shop nav, product grid above the fold, clear shipping/order promise.
